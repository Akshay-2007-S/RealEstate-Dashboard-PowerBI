# Real Estate Market Trends Dashboard

This Power BI dashboard analyzes property sales, demand, and pricing trends.  
Created during my internship with **CodeAlpha**, inspired by data practices from **Microsoft** and **Google**.

## Features
- KPI Cards for property status  
- Line & Column charts for trends  
- Donut chart for department-wise breakdown  
- Table with conditional formatting  
- Clean design with Canva background and icons  

## How to Open
Download the `.pbix` file and open it in **Microsoft Power BI Desktop**.
